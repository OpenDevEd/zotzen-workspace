# zotzen-workspace

Run
```
git clone git@github.com:OpenDevEd/zotzen-workspace.git
```
or
```
git clone https://github.com/OpenDevEd/zotzen-workspace.git
```

Then:
```
cd zotzen-workspace
./workspace.pl --clone
./workspace.pl --help
```

You need to have the File::Slurp library. On Ubuntu:
```
sudo apt install libfile-slurp-perl
```
or use `cpan`.
